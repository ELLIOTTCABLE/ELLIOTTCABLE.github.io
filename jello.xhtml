  • Jello

    Mac OS X pasteboard watcher and copy/paste manager

    Jello is a light and clean library for dealing with the Mac OS X pasteboard through the pbcopy and pbpaste commands. It also provides a binary that you can run in the background, that will process all strings pasted to the clipboard however you like - includes examples for shortening copied urls automatically as well as fixing ‘broken’ GrabUp URIs for you.
